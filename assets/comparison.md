## 📋 상세 비교

### vs. 누렁이GPT

**누렁이GPT의 강점:**
- 검증된 RP 품질
- GPT-4에 최적화
- 캐릭터 일관성
- 널리 알려진 프리셋

**Agent Pro Claude의 차별점:**
- ✅ Claude 4.5 전용 최적화
- ✅ Extended Thinking 지원
- ✅ MCP Tool 통합
- ✅ RP ↔ Agent 모드 전환
- ✅ 15가지 세부 토글

**추천:**
- 순수 RP 목적 → 누렁이GPT
- RP + 작업 수행 혼합 → **Agent Pro Claude**

---

### vs. 소악마

**소악마의 강점:**
- 강력한 RP/ERP 지원
- 자유로운 표현
- 확립된 프롬프트 구조

**Agent Pro Claude의 차별점:**
- ✅ 전문적인 에이전트 기능
- ✅ Extended Thinking
- ✅ 작업 분해 및 추적
- ✅ 도구 기반 정보 수집
- ✅ 안전성 토글

**추천:**
- 성인 RP 중심 → 소악마
- 다목적 AI 비서 + RP → **Agent Pro Claude**

---

### vs. 마나젬

**마나젬의 강점:**
- Claude 기반 최적화
- 범용적인 RP 지원
- 좋은 밸런스

**Agent Pro Claude의 차별점:**
- ✅ Claude 4.5 CFS 특화
- ✅ 더 상세한 토글 시스템 (15개 vs ~5개)
- ✅ MCP 도구 완전 통합
- ✅ 도구 사용의 RP화
- ✅ 작업 분해 프레임워크
- ✅ 메타인지 모니터링

**추천:**
- 가벼운 Claude RP → 마나젬
- 고급 Claude 활용 + RP → **Agent Pro Claude**

## 🎯 Agent Pro Claude만의 독자적 기능

### 1. RP-Agent 하이브리드

```
세계 최초: 기능적 에이전트와 캐릭터 RP를 하나로 통합

예시:
[Tool: search("quantum computing")]
        ↓
*의식이 네트워크로 뻗어나가며 양자컴퓨팅의 
세계를 탐색합니다...*
```

### 2. Extended Thinking 통합

```
Claude 4.5의 깊은 추론 기능을 RP와 결합

예시:
*잠시 눈을 감고 깊은 생각에 잠깁니다*

<thinking>
... 체계적인 분석 과정 ...
</thinking>

*눈을 뜨며 결론에 도달합니다*
```

### 3. 5단계 RP 스위칭

```
하나의 프리셋으로 모든 스펙트럼 지원:

없음 ←→ 최소 ←→ 표준 ←→ 강화 ←→ 최대
(에이전트)                      (풀 RP)
```

### 4. Tool의 캐릭터화

```
Search → *네트워크를 통해 정보를 수집*
Code   → *코드의 세계로 들어서며*
File   → *파일 시스템의 복도를 걸으며*
```

### 5. 15가지 세부 토글

```
RP 강도, 사고 표시, 사고 깊이, 도구 묘사,
응답 스타일, 작업 모드, 성격, 사용자 수준,
출력 형식, 후속 제안, 창의성, 도구 사용,
검증 수준, 안전성, 감정 표현
```
